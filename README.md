# 🍽️ Zomato Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Zomato dataset to uncover insights into customer ordering behavior, restaurant trends, and pricing patterns.
The goal is to transform raw data into meaningful insights that can help businesses make data-driven decisions.

---

## 🎯 Objectives

* Understand customer ordering patterns
* Identify popular cuisines and locations
* Analyze relationship between **cost, rating, and votes**
* Detect peak ordering times
* Provide insights for business growth strategies

---

## 📊 Dataset Information

The dataset contains information about:

* Restaurant names and locations
* Cuisines offered
* Average cost for two
* Ratings and votes
* Online ordering and table booking availability

📁 File: `zomato.csv`

---

## 🧹 Data Preprocessing

* Handling missing values
* Removing duplicates
* Converting data types
* Cleaning inconsistent entries

---

## 📈 Exploratory Data Analysis

Key analysis performed:

* 📌 Distribution of ratings
* 📌 Cost vs Rating analysis
* 📌 Online order availability impact
* 📌 Most popular cuisines
* 📌 Location-wise restaurant analysis

---

## 📉 Data Visualization

Used graphs to understand patterns:

* Bar charts
* Count plots
* Heatmaps
* Pie charts

Libraries used:

* Matplotlib
* Seaborn

---

## 🧠 Key Insights

* Most customers prefer restaurants with **ratings above 4.0**
* Online ordering significantly increases restaurant reach
* Certain cuisines dominate demand (e.g., North Indian, Chinese)
* Mid-range priced restaurants receive higher engagement
* Urban areas show higher order density
